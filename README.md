# Code Impact Training

A training and assessment repository for building professional development skills using Claude AI and modern dev tooling.

## Overview

This repo documents hands-on practice with real-world developer workflows — writing clean code, managing version control, and using AI-assisted development tools effectively.

## Skills Practiced

- Git & GitHub workflows (branching, committing, pushing, pull requests, merging)
- Using Claude Code as an AI coding assistant in VS Code
- Professional commit conventions and code organization
- Environment configuration and tooling setup
- TypeScript fundamentals

## Tools & Versions

- Node.js v24.14.1
- npm 11.11.0
- Git 2.52
- Claude Code 2.1.89
- VS Code
